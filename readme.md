# 📚 Study Buddy — AI Powered Tutor Chatbot

An intelligent AI study assistant built with TRAE IDE and MiniMax M2.7 model.

## 🚀 Features
- 🤖 Powered by MiniMax M2.7 AI Model
- 🧠 Deep Learning Mode — teaches the "why" behind concepts
- 💬 Supports Hindi & English both
- 😊 Emotional Intelligence — detects student mood
- 📝 Special Modes: exam mode, deep dive, explain like I'm 5
- 💾 Chat History saved in localStorage
- ✨ Premium dark UI with animations
- 📋 Code blocks with copy button

## 🛠️ Built With
- TRAE IDE (Builder AI)
- MiniMax M2.7 (via Puter.js)
- HTML, CSS, JavaScript

## 🎯 Special Commands
| Command | What it does |
|---|---|
| `exam mode` | Rapid fire Q&A practice |
| `deep dive` | Full detailed explanation |
| `explain like I'm 5` | Super simple language |
| `quick answer` | Short crisp answer |
