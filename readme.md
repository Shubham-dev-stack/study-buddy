## 🌟 What is Study Buddy?

Study Buddy is an intelligent AI-powered chatbot tutor that helps students understand any subject — anytime, anywhere. Whether you're stuck on a math problem at midnight or need a quick revision before an exam, Study Buddy is always there for you.

> 💡 Built at **TRAE x MiniMax Workshop, New Delhi 2026**

---

## ✨ Features

| Feature | Description |
|---|---|
| 🤖 **MiniMax M2.7 AI** | Powered by one of the most advanced AI models |
| 🧠 **Deep Learning Mode** | Teaches the "why" behind every concept |
| 💡 **Step-by-Step Thinking** | Breaks down complex problems into simple steps |
| 😊 **Emotional Intelligence** | Detects if you're frustrated or confident |
| 🌐 **Hindi + English** | Replies in whichever language you write in |
| 💾 **Chat History** | All conversations saved in localStorage |
| 🎨 **Premium Dark UI** | Glassmorphism design with smooth animations |
| 📋 **Code Blocks** | Syntax highlighted with one-click copy button |
