# 📚 Study Buddy — AI Powered Tutor Chatbot

An intelligent AI study assistant built with TRAE IDE and MiniMax M2.7 model.

## 🚀 Features
- 🤖 Powered by MiniMax M2.7 AI Model
- 🧠 Deep Learning Mode — teaches the "why" behind concepts
- 💬 Supports Hindi & English both
- 😊 Emotional Intelligence — detects student mood
- 📝 Special Modes: exam mode, deep dive, explain like I'm 5
- 💾 Chat History saved in localStorage
- ✨ Premium dark UI with animations
- 📋 Code blocks with copy button

## 🛠️ Built With
- TRAE IDE (Builder AI)
- MiniMax M2.7 (via Puter.js)
- HTML, CSS, JavaScript

|---|---|

## 🌟 What is Study Buddy?

Study Buddy is an intelligent AI-powered chatbot tutor that helps students understand any subject — anytime, anywhere. Whether you're stuck on a math problem at midnight or need a quick revision before an exam, Study Buddy is always there for you.

> 💡 Built at **TRAE x MiniMax Workshop, New Delhi 2026**

---

## ✨ Features

| Feature | Description |
|---|---|
| 🤖 **MiniMax M2.7 AI** | Powered by one of the most advanced AI models |
| 🧠 **Deep Learning Mode** | Teaches the "why" behind every concept |
| 💡 **Step-by-Step Thinking** | Breaks down complex problems into simple steps |
| 😊 **Emotional Intelligence** | Detects if you're frustrated or confident and responds accordingly |
| 🌐 **Hindi + English** | Replies in whichever language you write in |
| 📝 **Markdown Formatting** | Beautiful bold, bullets, and code block rendering |
| 💾 **Chat History** | All conversations saved — never lose your learning |
| 🎨 **Premium Dark UI** | Glassmorphism design with smooth animations |
| 📋 **Code Blocks** | Syntax highlighted code with one-click copy button |
| ⚡ **Lightning Fast** | Instant responses with animated typing indicator |

## 🚀 How to Run

1. Clone the repo
```bash
git clone https://github.com/Shubham-dev-stack/study-buddy.git
```

2. Open in TRAE or VS Code

3. Right click `index.html` → **Open with Live Server**

4. Start learning! 🎉

## 🧠 How the AI Works
```
Student types question
        ↓
MiniMax M2.7 receives question + full conversation history
        ↓
AI thinks step by step (Deep Learning Mode)
        ↓
Detects student emotion and adapts tone
        ↓
Returns formatted, friendly, encouraging answer
        ↓
Student learns better! 🎓
```
